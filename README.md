# plaxa-TUI-player
## this program runs off python and works only with linux terminals...

### SETUP 
python <br>
```pip install syncedlyrics deep-translator``` <br>

Linux

Arch <br>
```sudo pacman -S playerctl cava``` <br>

Fedora <br>
```sudo dnf install playerctl cava``` <br>

Debian/ubuntu/Mint/Pop!_os <br>
```sudo apt install playerctl cava```

REQUIREMENTS <br>
you must be using pipewire and the above installed
